# ARPMAN — Ableton Live Extensions

Free extensions for Ableton Live by **ARPMAN / WireDrifter**.  
Right-click context menus — no setup, no config, just install and go.

📺 [WireDrifter on YouTube](https://youtube.com/@WireDrifter)

---

## Extensions

### 🎹 MIDI Device Lookup
Browse CC and NRPN parameters for 399 hardware devices — no manual PDFs needed.  
Search by manufacturer or device name, filter parameters, copy CC numbers to clipboard.  
[Download](https://github.com/jdaehler/abletonliveextensions/releases/download/midi-device-lookup/midi-device-lookup.ablx) · Right-click any MIDI or Audio track

### 🎛️ Set Info
Quick overview of your current Live set — tempo, active scale, track count, scenes.  
[Download](https://github.com/jdaehler/abletonliveextensions/releases/download/set-info/set-info.ablx) · Right-click any MIDI or Audio track

### 🎼 Scale Filter
Remove or snap MIDI notes to the active Live scale.  
[Download](https://github.com/jdaehler/abletonliveextensions/releases/download/scale-filter/scale-filter.ablx) · Right-click any MIDI clip

### 📊 Clip Info
Statistics for a MIDI clip — note count, velocity range, pitch distribution.  
[Download](https://github.com/jdaehler/abletonliveextensions/releases/download/clip-info/clip-info.ablx) · Right-click any MIDI clip

### 🎨 Clip Colors
Colorize clips on a track by family, mode (fixed / random / cycle), and type.  
[Download](https://github.com/jdaehler/abletonliveextensions/releases/download/clip-colors/clip-colors.ablx) · Right-click any MIDI or Audio track

### 🔧 Note Forge
MIDI note generator — configure pattern, rhythm, and velocity, write directly to a clip.  
[Download](https://github.com/jdaehler/abletonliveextensions/releases/download/note-forge/note-forge.ablx) · Right-click any MIDI clip

### 👾 MIDI Invaders
Space Invaders played with your clip's active scale notes as targets.  
[Download](https://github.com/jdaehler/abletonliveextensions/releases/download/midi-invaders/midi-invaders.ablx) · Right-click any MIDI clip

### 🎧 BBC SFX Browser
Search and download BBC Sound Effects directly into your session — as a clip or into Simpler.  
Personal and educational use only (BBC RemArc licence).  
[Download](https://github.com/jdaehler/abletonliveextensions/releases/download/bbc-sound-effects/bbc-sound-effects.ablx) · Right-click any Audio track or Simpler

---

## Install

1. Download the `.ablx` file for the extension you want
2. Open Ableton Live → Preferences → Extensions
3. Drag the `.ablx` file into the Extensions window
4. Restart Live if prompted

All extensions show up in the **right-click context menu** on the relevant track or clip.
