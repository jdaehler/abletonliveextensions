## Scale Filter

**GitHub**
- Tag: `scale-filter`
- Release Title: `Scale Filter`
- File: `scale-filter.ablx`
- Discord Tags: `Utility` `Destructive`

**Discord Post**
```
🎼 Scale Filter

Two MIDI clip tools via right-click:

	• Remove — deletes all notes outside the active Live scale
	• Snap to Scale — moves out-of-scale notes to the nearest scale note (no notes lost)
	• Update check — shows a notification when a new version is available

Live's built-in scale mode only prevents wrong notes while recording — it doesn't touch notes already in a clip. Scale Filter works on existing clips and cleans them up after the fact.

Uses whatever scale you have set in Live's View → Scale panel.

Install: drag the .ablx file into Preferences → Extensions

## MIDI Invaders

**GitHub**
- Tag: `midi-invaders`
- Release Title: `MIDI Invaders`
- File: `midi-invaders.ablx`
- Discord Tags: `Generative` `Experimental`

**Discord Post**
```
👾 MIDI Invaders

A Space Invaders game that writes MIDI notes directly into your clip — right inside Ableton Live. Right-click any MIDI clip to start. Each column = one 16th note step, each row = a pitch from your current scale — shoot an invader to place that note.

	• Fully scale-aware — uses whatever key/scale is set in Live
	• MONO mode (default): one note per step, you choose the pitch by which invader you shoot
	• POLY mode: multiple pitches per step (chords)
	• APPEND / REPLACE toggle — layer patterns or start fresh
	• Clip gets named automatically (e.g. MI-C-Major)
	• Enemy fire keeps you on your toes 🔴
	• Update check — shows a notification when a new version is available

Install: drag the .ablx file into Preferences → Extensions

	• Update check — shows a notification when a new version is available
